## Khái niệm: Trung bình và Trung vị

### Trung bình (Mean)

[Giải thích trung bình là gì, khi nào dùng đúng]

$$
\bar{x} = \frac{x_1 + x_2 + ... + x_n}{n} = \frac{\sum_{i=1}^{n} x_i}{n}
$$

Trong đó:
- $\bar{x}$ (x-bar) là giá trị trung bình
- $x_i$ là từng giá trị riêng lẻ
- $n$ là tổng số quan sát

### Trung vị (Median)

[Giải thích trung vị, khi nào dùng đúng]

### Khi nào dùng cái nào?

[Quy tắc thực tế: phân phối lệch → trung vị; phân phối chuẩn → trung bình]

### Ví dụ với Python

```python
import pandas as pd
import numpy as np

# Dữ liệu đơn hàng trà sữa (nghìn đồng)
don_hang = [35, 42, 38, 45, 40, 500, 38, 36, 42, 39]

trung_binh = np.mean(don_hang)
trung_vi   = np.median(don_hang)

print(f"Trung bình: {trung_binh:.0f}k")  # bị kéo lên bởi đơn 500k
print(f"Trung vị:   {trung_vi:.0f}k")    # phản ánh thực tế hơn
```
